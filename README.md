# Admin-Panelli-Blog-Sitesi

Bu proje PHP ile yapılmıştır. Admin Paneli bulunmaktadır.

Nasıl Çalıştırılır ?
 
Adım 1 : XAMPP'de Apache ve MySQL'i aktif edin

Adım 2 : Habil ÇAKIR - Blog.rar dosyasını indirin ve ana klasörü kopyalayın

Adım 3 : C:\xampp\htdocs dizinine yapıştırın 

Adım 4 : Tarayıcınızı açın  ve "http:// localhost  / phpmyadmin /" URL'sine gidin

Adım 5 : Ardından veritabanları sekmesine tıklayın

Adım 6 : "habilcakir" adında bir veritabanı oluşturun ve ardından içe aktar sekmesine tıklayın

Adım 7 : Dosyaya gözat'a tıklayın ve habilcakir.sql dosyasını seçin ve yükleyin 

Adım 8 : Bir tarayıcı açın ve "http: // localhost " URL'sine gidin

Adım 9 : Oturum aç kısmından admin paneline erişebilirsiniz.
